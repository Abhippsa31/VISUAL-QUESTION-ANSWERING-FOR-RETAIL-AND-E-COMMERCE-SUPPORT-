# VISUAL-QUESTION-ANSWERING-FOR-RETAIL-AND-E-COMMERCE-SUPPORT
# 🍎 Smart Food Product VQA & Health Comparison

An AI-powered food analysis system using:

- BLIP Image Captioning
- BLIP Visual Question Answering (VQA)
- EasyOCR for label extraction
- Rule-based health scoring
- Text-to-Speech output
- Gradio Web Interface

---

## 🚀 Features

✅ Image Captioning  
✅ OCR Text Extraction  
✅ Health Score Calculation  
✅ Side Effects & Alternatives  
✅ Product Comparison  
✅ Visual Question Answering  
✅ Audio Output (TTS)  

---

## 🧠 Models Used

- Salesforce BLIP Image Captioning
- Salesforce BLIP VQA
- EasyOCR

---

## 📦 Installation

```bash
git clone https://github.com/your-username/smart-food-vqa.git
cd smart-food-vqa

pip install -r requirements.txt
```

---

## ▶️ Run the App
```bash
python app.py
```

It will launch a Gradio interface in your browser.

---

## 📷 How It Works
1. Upload food image(s)
2. OCR extracts label text
3. BLIP generates description
4. Health score is calculated
5. Optional question answering
6. Optional product comparison
7. Audio explanation generated
